Primeira opção: 

Usar a versão a cima de 20. Estou usando a versão 24 pelo nvm

comando para criar o projeto: 

* npm create vite@latest dashboard -- --template react

Acessar a pasta e instalar dependências:

* cd 20-dashboard
* npm install

Rodar o projeto localmente: 

* npm run dev

link do video do projeto: https://www.youtube.com/watch?v=ibOz6Lz40xU&list=PLwP3cL-MKVkNM28X96Dhc3BLMhtUktiik&index=1

A partir daqui irei utilizar a biblioteca Mui, que eu consigo acessando o mui.com. O link: https://mui.com

Esse mui tem uma serie de repositórios pre prontos para serem utilizados no react. é basicamente uma biblioteca de componentes parecido com o quasar.

Comando para instalar o MUI, necessário que seja qualquer versão a cima de 18 do React: 

* npm install @mui/material @emotion/react @emotion/styled

Caso depois de instalar de o erro a baixo utilize o comando a seguir para baixar os ícones da biblioteca mui:

Para instalar novos ícones é só utilizar o comando a seguir:

* npm install @mui/icons-material

Para poder utilizar o sistema de router do react é necessario instalar com o comando a baixo:

* npm install react-router-dom
